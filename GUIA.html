PASSO 1 de 4

🗄️ Criar o banco de dados (Supabase)

Grátis~5 min

1. Acesse supabase.com e clique em "Start your project"

2. Crie uma conta (pode usar Google/GitHub)

3. Clique em "New project", escolha um nome (ex: amigosnet) e selecione a região South America (São Paulo)

4. Aguarde ~1 minuto para o projeto criar

5. No menu lateral, clique em "SQL Editor"

6. Cole o conteúdo do arquivo supabase_setup.sql e clique em "Run"

⚡ Isso cria a tabela de localizações e ativa o modo tempo real!

7. Vá em Project Settings → API e copie:

  • Project URL → começa com https://xxxxx.supabase.co
  • anon public key → chave longa começando com eyJ...
PASSO 2 de 4

✏️ Colocar suas Keys no app

~2 min

Abra o arquivo index.html num editor de texto (Notepad, VSCode, etc.)

Procure essas duas linhas lá em baixo no <script>:

const SUPABASE_URL = window.ENV_SUPABASE_URL || 'COLE_SUA_URL_AQUI'; const SUPABASE_ANON = window.ENV_SUPABASE_ANON || 'COLE_SUA_ANON_KEY_AQUI';

Substitua pelos seus valores:

const SUPABASE_URL = window.ENV_SUPABASE_URL || 'https://SEU_ID.supabase.co'; const SUPABASE_ANON = window.ENV_SUPABASE_ANON || 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';

🔒 A anon key é segura para ficar no frontend — ela só permite ler/gravar localizações, nada mais.

PASSO 3 de 4

🌐 Hospedar no Netlify (link público)

Grátis~3 min

1. Acesse netlify.com e crie uma conta grátis

2. Na dashboard, arraste a pasta amigosnet inteira para a área indicada:

📁 Arraste a pasta que contém: index.html + netlify.toml

3. Aguarde ~30 segundos para o deploy

4. O Netlify vai te dar um link como:

https://meu-amigosnet.netlify.app

5. (Opcional) Clique em "Domain settings" para personalizar o nome da URL!
